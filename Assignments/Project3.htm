To copy the sql dbase 'base' to the sql dbase 'copied', there are below different approaches.

1. Take a backup of 'base' databse as .bak file and then restore to a new database named 'copied'.
2. Generate scripts of all objects such as Table, SP, View etc. of 'base' database including the data amd then create databse named 'copied' and run the script.
3. Using DBCC commands : DBCC CLONEDATABASE ('base', 'copied').
